# Movie/Book Finder 🎬📚

https://movie-finder-its-me-yo.vercel.app/

A simple web app to search for movies using the OMDb API.

## Features
- Search movies by title
- Displays title, poster, and year
- Responsive and visually appealing UI

## Tech Stack
- HTML, CSS, JavaScript
- OMDb API

## Deployment
- Static site, deployable on Vercel or Netlify
